HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The body element is divided up into thematic section elements.
  • Attributes are added inside the opening element, never the closing element.
  • Text heading elements should appear in logical semantic sequence with only one <H1> tag on a page.
  • The header element includes information that can be duplicated on every page such as navigation menus.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git pull origin main: copies the main branch into the local folder
  • git clone <respository> copies the repo into the local folder

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a container for multiple values in order
  • A for loop can select each item in an array one at a time
  • A function is a block of code that can be reused in a program
